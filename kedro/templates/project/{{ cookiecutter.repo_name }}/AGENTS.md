# Agent Guidelines

This is a [Kedro](https://docs.kedro.org) project. These guidelines help AI coding assistants work with it correctly.

## Catalog configuration

When editing files in `conf/` (especially `catalog.yml`), read [.agents/skills/catalog-config/SKILL.md](.agents/skills/catalog-config/SKILL.md) for dataset type naming, argument conventions, factory patterns, and credential handling.
